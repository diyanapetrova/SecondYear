To run ttable2:
./ttable2 <flag> <number of bool vars> <expression> <expression>

flag:
'n' - basic
'c' - print categories of the expression 
'd' - print the table for the second expression where the vars values make the first one true
'e' - print truth table for exp1 = exp2 and statement whether they are equvalece
's' - solver print only the line that is true

number of vars:
1-26
